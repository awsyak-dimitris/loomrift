# Loomrift — Product Vision

## What It Is
A visual operating system for teams, projects, people and workflows.
Not a dashboard. Not a CRM. A living map of your professional universe.

## Core Metaphor
Every entity is a node. Every relationship is an edge. The graph is the interface.

## Design References
- **Linear** — calm, focused, premium feel
- **Cursor IDE** — technical workspace aesthetic
- **Figma** — inspector panel, right-click context
- **Arc Browser** — spatial organization
- **Obsidian Graph** — knowledge graph visualization
- **Unreal Engine Editor** — node-based visual systems
- **Palantir Gotham** — data relationship mapping
- **Raycast** — command palette, keyboard-first

## Principles
1. **Graph First** — the graph is always the hero, never hidden behind modals
2. **Calm Interface** — dark, quiet, restrained glow, no visual noise
3. **Premium Desktop Feeling** — feels like a native pro app, not a web dashboard
4. **Keyboard + Context Menu** — fast interactions without leaving the graph
5. **Inspector over Dialogs** — right panel for details, not popup forms

## Target Users
- Founders managing multiple projects
- Technical leads tracking team structure
- Creative directors overseeing production pipelines
- Anyone who thinks spatially about their work

## Current State
Single-file HTML app with Three.js 3D graph, localStorage persistence,
avatar upload, alert markers, AI summary, multi-root nodes.

## Future Direction
- Google Contacts/Calendar/Drive integration
- Real-time collaboration
- Command palette (⌘K)
- Figma-quality inspector panel
- Export/import/share graphs
